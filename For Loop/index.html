    Question 14: Write a program that prints number from start of the array to desired stop value. Given array:

    if user gives 34 as input value print 12, 45, 3, 22, 34

    Solution: const scores = [12,45,3,22,34,50]; stopValue = +prompt('Enter stop value from the array'); const div = document.querySelector('.div') for(let i =0;i <= scores.length;i++){ div.innerHTML += `
    ${scores[i]}
    ` if(scores[i] === stopValue){ break; } } if(scores[i] !== stopValue){ div.innerHTML += `
    Your value is not available in the array
    ` }
